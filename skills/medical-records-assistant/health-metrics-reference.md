# 常见体检指标参考

通用参考范围，具体以体检报告上的参考值为准（不同医院、不同检测方法可能有差异）。

## 血常规

| 指标 | 参考范围 | 偏高可能 | 偏低可能 |
|------|----------|----------|----------|
| 白细胞 (WBC) | 3.5-9.5 x10^9/L | 感染、炎症 | 免疫力低 |
| 红细胞 (RBC) | 男 4.3-5.8 / 女 3.8-5.1 x10^12/L | 脱水、慢性缺氧 | 贫血 |
| 血红蛋白 (Hb) | 男 130-175 / 女 115-150 g/L | 同上 | 贫血 |
| 血小板 (PLT) | 125-350 x10^9/L | 感染、炎症 | 出血风险 |

## 血糖

| 指标 | 参考范围 | 说明 |
|------|----------|------|
| 空腹血糖 | 3.9-6.1 mmol/L | >6.1 需复查，>7.0 考虑糖尿病 |
| 糖化血红蛋白 (HbA1c) | <6.0% | 反映近 2-3 个月血糖水平 |

## 血脂

| 指标 | 理想范围 | 说明 |
|------|----------|------|
| 总胆固醇 (TC) | <5.18 mmol/L | >6.22 为高胆固醇 |
| 甘油三酯 (TG) | <1.7 mmol/L | >2.26 为偏高 |
| LDL-C（坏胆固醇） | <3.37 mmol/L | 越低越好，心血管核心指标 |
| HDL-C（好胆固醇） | >1.04 mmol/L | 越高越好 |

## 肝功能

| 指标 | 参考范围 | 说明 |
|------|----------|------|
| ALT（谷丙转氨酶） | 9-50 U/L | 偏高提示肝损伤 |
| AST（谷草转氨酶） | 15-40 U/L | 偏高提示肝/心肌损伤 |
| GGT（y-谷氨酰转肽酶） | 10-60 U/L | 偏高可能与饮酒、脂肪肝有关 |

## 肾功能

| 指标 | 参考范围 | 说明 |
|------|----------|------|
| 肌酐 (Cr) | 57-111 umol/L | 偏高提示肾功能下降 |
| 尿酸 (UA) | 男 208-428 / 女 155-357 umol/L | 偏高有痛风风险 |
| 尿素氮 (BUN) | 3.1-8.0 mmol/L | 偏高提示肾功能下降 |

## Apple Watch 指标参考

| 指标 | 良好范围 | 说明 |
|------|----------|------|
| 静息心率 | 50-70 bpm | 越低越好（运动员常 <50） |
| HRV (SDNN) | >50 ms | 越高越好，反映自主神经调节能力 |
| VO2Max | 男 >40 / 女 >35 mL/min*kg | 心肺耐力，随年龄下降 |
| 血氧 | 95-100% | <90% 需就医 |
| 深睡占比 | 15-25% 总睡眠 | Apple Watch 检测偏保守 |

> 本文件仅供参考，不构成医疗建议。具体健康问题请咨询医生。
