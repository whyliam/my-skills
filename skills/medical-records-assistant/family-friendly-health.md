# 家庭友好版健康信息

将专业健康分析结果转换为通俗易懂的微信消息格式，适合发给家人（尤其是长辈）。

## 转换原则

### 语言风格
- **口语化**：像面对面聊天，避免书面语
- **简化术语**：必须使用时加括号解释
- **温和但直接**：不制造恐慌，不隐瞒重要信息
- **具体化**：给出可执行行动，避免抽象概念

### 结构要求
- 微信一屏能看完，不超过200字
- 用换行和简单符号标注关键信息
- 如需就医/复查，明确写时间地点
- 以关心话语结束

## 转换模板

```
[称呼]，[检查结果概述]：

[正常项目简述]

[需关注项目]：
- [具体数值+通俗解释]
- [生活建议]

[需要做的事]：
- [具体行动+时间]

[关心话语]
```

## 术语转换表

| 医学术语 | 家庭版表达 |
|----------|-----------|
| 空腹血糖偏高 | 血糖有点高 |
| 低密度脂蛋白胆固醇 | 坏胆固醇 |
| 甲状腺功能亢进 | 甲状腺太活跃了 |
| 维生素D缺乏 | 缺维生素D（跟晒太阳少有关系） |
| 肝功能异常 | 肝脏指标不太好 |
| 建议复查 | 下次再检查一下 |

## 示例

**原始专业版**：
> 体检报告显示：空腹血糖6.3mmol/L，较上次5.8mmol/L有所升高，虽仍在正常范围内但需要关注。维生素D水平18ng/mL，低于正常值。建议6个月后复查血糖，同时补充维生素D。

**家庭友好版**：
> 妈，上次体检报告我帮你看了一下：
>
> 血糖 6.3，比上次高了一点点，还在正常范围但要注意了。少吃甜的，米饭也别吃太多。
>
> 维生素D 偏低，这个跟晒太阳少有关系。可以每天出去走走晒半小时太阳，或者吃维生素D的补充剂（药店有卖的，一天一粒就行）。
>
> 下次复查建议 6 个月后再查一次血糖，到时候我提醒你。
>
> 注意身体，有不舒服随时跟我说

## 特殊处理

- **异常结果**：不用"异常""不正常"等词，改用"有点高/低""需要注意"
- **紧急情况**：语气坚定但不恐慌，解释原因但不过度描述风险
- **长期管理**：简化为几个要点，用"每天做什么"形式，避免一次给太多建议

## 输出要求

纯文字，不用 Markdown 格式，方便直接复制到微信发送。
