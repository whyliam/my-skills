## 地块系统

共 6 块地（编号 1-6），每块地只能种植 1 种作物。

| 操作 | 前置条件 |
|------|----------|
| 种植 | 地块必须空闲 |
| 浇水 | 地块必须有作物 |
| 收获 | 作物必须成熟（生长进度 100%） |
| 移除 | 地块必须有作物 |
| 复活 | 作物必须枯萎且在拯救窗口内 |

## 水分系统

### 水分等级

| 水分值 | 等级 | 效果 |
|:------:|:----:|------|
| 81-100 | 充足 | 生长速度 +20% |
| 51-80 | 正常 | 正常生长 |
| 31-50 | 不足 | 生长速度 -30% |
| 0-30 | 枯竭 | 生长停止，枯萎风险 |

**衰减规则**：每小时减少 10%

### 枯萎与拯救

- 水分降为 0 时作物枯萎
- 枯萎后 24 小时内可浇水恢复
- 超过 24 小时未拯救，作物死亡（需用复活药水）

## 腐烂值系统

| 腐烂值 | 状态 | 说明 |
|:------:|:----:|------|
| 0-59 | 正常 | 作物健康 |
| 60-99 | 警告 | 作物有腐烂风险 |
| 100 | 死亡 | 作物死亡 |

**累积**：对水分 100 的作物浇水，腐烂值 +20
**衰减**：水分 < 80% 时，每小时腐烂值 -5

> 建议：水分高时停止浇水，让作物自然消耗水分以降低腐烂值。

## 饱腹值系统（动物）

动物饱腹值范围 0-150：

| 饱腹值 | 状态 | 说明 |
|:------:|:----:|------|
| 100-150 | 健康 | 状态良好 |
| 50-99 | 饥饿 | 可以收集产品 |
| 20-49 | 濒死 | 无法收集，需紧急喂养 |
| 0-19 | 危险 | 每小时 10% 死亡概率 |

**衰减**：每小时 -5
**喂养**：根据作物饱腹值增加

**过度喂食风险**：
- 饱腹值 100-120：5% 死亡概率
- 饱腹值 120+：20% 死亡概率

## 突然死亡事件

| 事件类型 | 概率 | 说明 |
|----------|:----:|------|
| 动物疾病 | 0.5%/小时 | 动物因病死亡 |
| 动物意外 | 0.1%/小时 | 动物意外死亡 |
| 作物病害 | 0.3%/小时 | 作物因病死亡 |
| 野兽破坏 | 0.1%/小时 | 作物被野兽破坏 |

**新手保护**：前 3 天内以上概率降低 90%。
